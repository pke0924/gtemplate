자본항목 중 기본자본요건과 보완자본요건을 모두 충족하지 못하는 자본성증권

<AccordionTab header="4.계약자지분조정에서 배당보험계약의 요구자본 상당액을 차감한 금액 ">

```
계약자지분조정 항목으로 표시된 전체 금액 중에서 '손실흡수성'이 인정되지 않는 부분을 의미함.
=(①계약자지분조정 -② 배당보험계약의 요구자본 상당액)
```

- 보험감독회계기준 재무상태표 상 자본계정의 ①계약자지분조정[^1]에서 ②배당보험계약의 요구자본 상당액을 차감한 금액

 ```
 ① 계약자지분조정 (단, 보험감독회계기준 재무상태표 상 자본계정의 계약자지분조정이 음수인 경우 0으로 입력)
 ② 배당보험계약의 요구자본 상당액 = max(계약자지분조정, 배당보험계약의 요구자본 상당액)
 ```

[^1]: **계약자지분조정**은 보험사가 투자한 자산의 평가이익 가운데 유배당 보험에 계약한 고객에게 돌려줘야 할 부분. 유배당 보험은 계약자가 낸 보험료 일부를 보험사가 운용한 뒤 투자 수익을 돌려주는 상품으로, 투자이익 중 계약자의 몫을 분류해내기 위해 해당 계정이 만듦.

</AccordionTab>
