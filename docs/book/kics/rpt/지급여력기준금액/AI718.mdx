# 자본증권 명세표 (AI718)

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">
- ⒈ `구분(자본/부채 여부)`는 해당 자본증권이 건전성감독기준 재무상태표 상 자본항목인지 부채항목인지 구분하여 `자본` 또는 `부채`로 기입
- ⒉ `보통주 이외의 자본증권 분류`는 아래 표를 참고하여 해당되는 분류를 기입

    | 구분 | 자본증권 분류 |
    |:---:|:---:|
    | 자본 | 누적적 우선주, 비누적적 우선주, 신종자본증권, 기타 자본항목 |
    | 부채 | 상환 우선주, 후순위채무 |

- ⒊ `발행금액`: 신종자본증권은 감독규정 제7-9조제9호의 규정에 의한 신종자본증권 발행액, 후순위채무는 감독규정 제7-9조제5호의 규정에 의한 후순위채무 발행액을 입력
- ⒋ `평가금액`: 부채항목(상환 우선주, 후순위채무)에 대해서만 입력.공정가치에 따라 인식된 금액으로 평가하며 [별표22]"Ⅱ.2-4"의 평가기준을 따름
- ⒌ `K-ICS 인정금액`: 보완자본에 대해서만 입력.	만기시 지급유예조항(Lock-in 조항)을 보유하지 않은 자본증권은 잔존만기(경제적 만기 포함)가 5년 미만인 시점부터 매년 보완자본 불인정금액을 차감
- ⒍ `계층분류 기준 충족(또는 미충족) 근거`는 ①가용성, ②지속성, ③후순위성, ④기타 제한의 부재 등 요건별로 상세히 기입할 것
