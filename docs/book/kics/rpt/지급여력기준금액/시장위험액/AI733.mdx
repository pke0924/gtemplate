# 시장위험액-주식위험 (AI733)

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">

- ⒈ `기본법` : 시행세칙 [별표22] <OverlayDemo id = "443"/> 에서 정한 산출방법에 따라 충격 전후 금액을 입력
- ⒉ `간편법` : 시행세칙 [별표22]의 <OverlayDemo id = "416_나"/>"⑴①"에 대해 <OverlayDemo id = "416_나"/>"⑶①"의 주식위험액 산출방법으로 측정한 금액을 입력
- ⒊ `일반계정` :생보 일반계정 및 손보 일반/특별(장기)계정 등 퇴직 및 변액을 제외한 모든 계정
- ⒋ 변액보험 부채의 충격 전·후 가치가 주식 유형별로 구분되지 않는 경우 변액보험 자산의 주식유형별 비중으로 배분 가능
- ⒌ 위험경감 & **문서화**된 잔존만기1년 미만 위험경감 파생상품은 충격 후 가치 산정시 위험경감 인정비율 만큼만 적용
    - `충격후 가치 = 충격전 가치 + 충격으로 인한 가치변동*(잔존만기비율 + (1-잔존만기비율)*80%)`
      <Cmt>
      - (예시) 기존가치 100, 잔존만기 3개월, 충격으로 인한 가치 +40% 변동시 충격후 가치 = 100 + 40*(0.25 + 0.75*80%)  = 134
      - 다만, 변액보험은 위험경감&문서화 인정시 위험경감비율을 100% 적용
      - (예시) 기존가치 100, 충격으로 인한 가치 +40% 변동시 충격 후 가치 = 100 + 40 = 140
      </Cmt>
- ⒍ 위험경감 & **비문서화**된 잔존만기1년 미만 위험경감 파생상품은 충격 후 가치 산정시 위험경감 인정비율 만큼만 적용.
    - `충격후 가치 = 충격전 가치 + 충격으로 인한 가치변동*잔존만기비율`
      <Cmt> 
      \* (예시) 기존가치 100, 잔존만기 3개월, 충격으로 인한 가치 + 40% 변동시 충격후 가치 = 100 + 40*0.25  = 110 
      </Cmt>
- ⒎ `우선주` : 보험사가 투자한 신종자본증권, 후순위채권 및 후순위대출 등의 자산 중 자본증권 계층분류기준의 기본자본 요건 또는 보완자본요건을 충족하는 자산을 포함
- ⒏ `시장성 있는 무형자산`은 기타주식에 포함
- ⒐ 단, K-ICS 계량영향평가시 간편법은 작성하지 않고, 기본법에만 기재
