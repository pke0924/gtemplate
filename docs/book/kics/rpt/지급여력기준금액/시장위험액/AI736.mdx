  - ⒈ `기본법` 적용 : 시행세칙 [별표22] <OverlayDemo id = "445"/>"Ⅳ.4-5"에서 정한 산출방법에 따라 외환위험에 노출된 현물 및 파생의 환율 충격 전후 금액과 가격변동위험액을 입력
  - ⒉ `간편법` 적용 : 시행세칙 [별표22]의<OverlayDemo id = "416_나"/> "⑴①"에 대해 <OverlayDemo id = "416_나"/>"⑶①"의 외환위험액액 산출방법으로 측정한 통화별 환율하락위험액, 환율상승위험액, 가격변동위험액을 입력

  - ⒏ 통화는 중복되지 않도록 기입
  - ⒐ 단, K-ICS 계량영향평가시 간편법은 작성하지 않고, 기본법에만 기재

<TabView>
    <TabPanel header="위험경감 100% 인정 파생상품" >
    위험경감파생상품 중 파생상품 잔존 만기가 1년 이상이거나, 파생상품의 잔존만기가 위험경감 대상의 잔존만기 이상인 경우
    </TabPanel>
    <TabPanel header="위험경감 100% 인정 파생상품 外 " >
      `갱신계획 비문서화` : 위험경감 100% 인정 파생상품에 해당하지 않고 갱신계획이 별도 관리되지 않는 파생상품.
      ```
        충격후 가치 = 충격전 가치 * (1-경감효과 인정 비율) + 충격 100% 반영시 가치 * 경감효과 인정 비율
        경감효과 인정 비율 = 잔존만기비율
      ```

      `갱신계획 문서화` : 위험경감 100% 인정 파생상품에 해당하지 않고 갱신계획이 별도 관리되는 파생상품.
      ```
          충격후 가치 = 충격전 가치 * (1-경감효과 인정 비율) + 충격 100% 반영시 가치 * 경감효과 인정 비율
          경감효과 인정 비율 = 잔존만기비율 + (1-잔존만기비율)*80%
      ```
    </TabPanel>
    <TabPanel header="가격변동위험액" >
     - – 잔존만기 1년 미만 외환위험 경감기법은 가격변동 위험액을 추가 적용한다.<br/> 갱신 계획 문서화 파생의 명목 계약금액에 1%(위험경감자산 계약만기 1년 이상) 또는 2%(위험경감자산 계약만기 1년 미만)를 곱하여 산출(원화환산)
      
      - – 가격변동 위험액 = 외화 위험경감기법 계약의 명목금액 * 측정시점 환율의 x%
        - ∙ x = 2 (위험경감기법의 계약만기가 1년 미만인 경우)
        - ∙ x = 1 (위험경감기법의 계약만기가 1년 이상인 경우)
        

    </TabPanel>
