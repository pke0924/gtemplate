# 시장위험액-금리위험 (AI732)

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">

- ⒈ `간편법 적용 간접투자기구`를 제외한 모든 항목은 시행세칙 [별표22] <OverlayDemo id = "442"/>에서 정한 산출방법에 따라 충격 전후 금액을 입력
- ⒉ `간편법 적용 간접투자기구` : 시행세칙 [별표22]의 <OverlayDemo id = "416_나"/>⑴①에 대해 <OverlayDemo id = "416_나"/>⑶①의 금리위험액 산출방법으로 측정한 위험액을 충격 전후 금액을 구분하여 기입
- ⒊ `후순위채권` : 보험회사가 발행한 후순위채권 중 지급여력금액 불인정 금액
- ⒋ `공동재보험` : 원보험사가 위험보험료 외에 저축보험료 등의 일부도 재보험사에 출재하고, 보험위험 외에 금리위험 등 다른 위험도 재보험사에 이전하는 보험
- ⒌ IRS 등 금리위험경감 목적의 파생상품이 K-ICS 위험경감기법으로 인정될 경우 `금리위험경감 인정 파생상품`에 입력하고, 인정되지 않을 경우 `금리위험경감 미인정 파생상품`에 입력
- ⒍ `기타파생상품` : 환파생상품 등 금리위험경감 목적이 아닌 파생상품을 입력
- ⒎ 단, K-ICS 계량영향평가시 간편법은 작성하지 않고, 기본법에만 기재
