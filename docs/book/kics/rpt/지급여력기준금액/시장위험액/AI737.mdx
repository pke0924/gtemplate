# 시장위험액-자산집중위험 (AI737)

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">

- ⒈ `K-ICS 시장위험액-자산집중위험`은 자산집중위험 한도를 초과한 거래상대방에 대한 익스포져를 입력
   - – 작성 대상 거래상대방 및 부동산이 주어진 양식을 초과할 경우 신용등급을 기준으로 합산하여 입력 가능
   - – 단, 자산집중위험 한도를 초과한 거래상대방 및 부동산이 없는 경우, `- ⒈ 거래상대방 관련 익스포져`는 자산집중위험 익스포져 기준 상위 5개, `- ⒉ 부동산`은 는 자산집중위험 익스포져 기준 상위 3개를 입력
- ⒉ `자산총계`, `운용자산` : 특별계정 퇴직연금 실적배당형 자산과 변액보험 자산 및 종속회사 중 업권별 요구자본 또는 총자산 8% 적용회사의 자산을 제외한 자산을 대상으로 함
- ⒊ `연결전 기준 위험액` : 연결전 지분 익스포져 기준으로 측정한 자산집중위험 중 거래상대방집중위험액
- ⒋ `장내 및 장외파생상품` : 기초자산이 자산집중위험액 측정대상에 해당하는 장내·외 파생상품
