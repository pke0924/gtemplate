# 비례성원칙(적용현황) (AI744)

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">

- ⒈ 10차 QIS에서는 연결산 기준으로 적용가능한 "Ⅴ. 기타 종속회사 요구자본" 간편법 적용을 희망하는 경우에만 작성
- ⒉ `K-ICS 비례성원칙(적용현황)`은 시행세칙 [별표 22]의 <OverlayDemo id ="416_다"/>에서 정한 절차에 따라 비례성원칙의 간편법을 적용하는 경우에 작성
- ⒊ `1. 간편법 적용 종속회사 ~ 6. 간편법 적용 종속회사`의 종속회사 회사명은 비례성원칙에 따라 간편법을 적용하여 요구자본을 산출하고자 하는 종속회사의 회사명을 기재
- ⒋ 비중요성 기준 세부현황은 시행세칙 [별표 22] <OverlayDemo id ="416_나"/>⑵의 비중요성 기준에 따라 작성
   - – `1. 간편법 적용 종속회사 ~ 6. 간편법 적용 종속회하`의 기준1~기준2는 시행세칙 [별표 22] <OverlayDemo id ="416_나"/>⑵⑤㈀ ~ ㈁ 의 기준에 따라 산출한 비중(%)을 기재
- ⒌ 간편법 적용현황은 시행세칙 [별표 22] <OverlayDemo id ="416_나"/>⑶의 간편법 적용방식에 따라 작성
   - – `Ⅴ. 기타 종속회사 요구자본`의 당분기말 요구자본은 시행세칙 [별표 22] <OverlayDemo id ="416_나"/> ⑶⑤에 따라 간편법을 적용하여 산출한 종속회사의 분기결산시점 요구자본을 단순 합산한 금액
   - – `1. 간편법 적용 종속회사 ~ 6. 간편법 적용 종속회사`의 당분기말 요구자본은 시행세칙 [별표 22] <OverlayDemo id ="416_나"/>⑶⑤에 따라 간편법을 적용하여 산출한 분기결산시점 요구자본
- ⒍ `입증결과 제출일`에는 시행세칙 [별표 22] <OverlayDemo id ="416_다"/>에 따라 해당 비례성원칙 충족 여부 입증 결과를 감독원에 제출한 일자를 기재
- ⒎ `최초 적용일`에는 해당 비례성원칙 적용 부문을 최초로 적용한 일자를 기재
- ⒏ `연장 적용일`에는 해당 비례성원칙을 최초로 적용한 후 3년이 경과하여 간편법을 계속 사용하고자 할 경우, 연장 적용일이 시작되는 일자를 기재 (단, 최초 적용한지 3년 미만의 경우 기재 불필요(공란으로 둠))
- ⒐ `최종 적용일`에는 보험회사가 해당 비례성원칙을 사용하고자 계획한 예상 최종 적용일을 기재
