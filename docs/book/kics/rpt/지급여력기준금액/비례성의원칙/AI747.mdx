# 비례성원칙(생명·장기손해보험리스크 하위위험 요구자본) (AI747)

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">

- ⒈ `K-ICS 비례성원칙(생명·장기손해보험리스크 상품그룹 기준)`은 시행세칙 [별표 22]의 <OverlayDemo id ="416_다"/>에서 정한 절차에 따라 <OverlayDemo id ="416_나"/>⑴③의 생명·장기손해보험리스크의 하위위험에 대한 요구자본에 대하여 간편법을 적용하는 경우에 작성
- ⒉ `Ⅰ. 직전년도 연결산시점 요구자본`은 시행세칙 [별표 22]의 "Ⅳ.2-2 ~ 2-7"에 따라 산출한 금액
- ⒊ `Ⅱ.1. 하위위험별 익스포져명`은 하위위험별 리스크 실질에 부합하는 항목을 보험회사가 선정하여 기재
- ⒋ `Ⅱ.2. 하위위험별 익스포져`는 '3)'에서 선정한 기준에 따라 직전년도 연결산시점 기준 금액을 집계
- ⒌ `Ⅲ. 간편법 적용항목에 대한 위험계수(fk)`는 하위위험별로 아래의 산식으로 산출
   - – 하위위험별 위험계수 = 직전년도 연결산시점 하위위험액 ÷ 하위위험별 직전년도 연결산시점 익스포져
- ⒍ `Ⅳ. 분기결산시점 익스포져`는 '3)'에서 선정한 기준에 따라 분기결산시점 기준 금액을 집계
- ⒎ `Ⅴ. 분기결산시점 요구자본 간편법`은 하위위험별로 아래의 산식으로 산출
   - – `분기결산시점 하위위험별 위험액` `= max(직전년도 연결산시점 하위위험액, 하위위험별 위험계수 × 하위위험별 분기결산시점 익스포져)`
