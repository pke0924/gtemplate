# 비지배지분 중 종속회사 요구자본의 비지배지분 상응액 (AI719)

- <OverlayDemo id ="418"/>
<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">

- ⒈ `업종유형`은 주석의 구분Code 참고
- ⒉ `요구자본 산출방법` : ①연결BS합산, ②업권별요구자본, ③비례성원칙에 따라 간편법 적용
- ⒊ 연결대상 비금융회사 경우 업종 유형은 공란, 수익증권 등 비지배지분이 자본계정이 아닌 부채 계정으로 계상될 경우 비지배지분 상응액 미산출
- ⒋ `비지배지분`은 그룹 가용자본에 포함된 종속회사에서 발행한 자본증권을 포함
- ⒌ 요구자본 산출방법에서  "②업권별 요구자본"을 선택한 경우 다음 사항을 참고하여 `지급여력기준금액`에 환산 후 금액을 입력
  - 요구자본은 해당 업권의 요구자본 수치 및 환산율을 적용(금융회사의 경우 아래 금융업권별 요구자본 테이블\<표5> 참조)
- ⒍ 요구자본 산출방법에서  "③비례성원칙에 따라 간편법 적용"을 선택한 경우 `지급여력기준금액`에 총자산 대비 환산한 금액을 입력
