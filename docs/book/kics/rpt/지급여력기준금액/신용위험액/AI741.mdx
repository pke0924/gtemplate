# 신용위험액(담보부자산) (AI741)

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">

- ⒈ `K-ICS 신용위험액(담보부자산)`은 시행세칙 [별표22] <OverlayDemo id ="451"/> ~ <OverlayDemo id ="453"/>에서 정한 산출방법으로 측정한 금액을 입력
- ⒉ `신용 익스포져` : 적격금융자산담보대출을 제외한 상업용부동산 담보대출 및 주택담보대출 익스포져를 대상으로 작성
- ⒊ 임대수익 독립형 상업용부동산 담보대출은 현행 LTV에 따른 위험량과 신용등급 및 유효만기별 위험계수에 따른 위험량을 비교 작은값을 사용할 수 있음
    - ※ `Min(LTV기준 계수, 신용등급기준 위험계수)`
- ⒋ 중·후순위 담보대출의 신용위험액은 선순위 담보대출을 합산하여 산출한 신용위험액에서 선순위 담보대출의 신용위험액을 차감한 값

<Accordion multiple activeIndex={[]}>
<AccordionTab header="<예시>">

LTV만 산출 가능한 상업용 부동산 담보대출의 경우 잔액기준 LTV로 위험계수 수준으로 판단하되, 익스포져는 공정가치를 기준으로 산출

- 담보가치 : 1000, 1순위대출(은행): 500 (평가시 480), 2순위대출(보험사): 400 (평가시 390)
- 총 대출 : 900 (LTV 90%)
- 선순위대출 : 500 (LTV 50%)
<Cmt>
