# 계리적가정 정합성 검증(예상-실제 차이) (AI714)

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">

- ⒈ 지배회사의 생명/장기손해보험계약을 대상으로 작성한다.
- ⒉ '`실제`'부분은 결산월을 포함한 이전 12개월간 발생한 실적을 기반으로 작성한다.
- ⒊ '`예상`'부분은 장래 현금흐름에서 최초 12개월분을 단순합산하여 작성한다. 장래현금흐름 산출시점은 결산월을 기준으로 이전 12개월로 한다.
- ⒋ `신계약 포함(A)`은 모든 보험계약을 대상으로 한다.
- ⒌ `신계약 제외(B)`는 A에서 결산월을 포함한 이전 12개월간 유입된 신계약의 실적을 차감하여 작성한다.<br/>
   단 직전 1년간의 신계약 실적을 집계하기 어려운 항목은 회사가 합리적으로 배분한 결과를 활용한다.
- ⒍ `예상(C)`을 작성하기 위한 장래현금흐름은 결산월말 기준 이전 12개월 시점 보유계약을 대상으로 한다.<br/>
   결산월말 보험부채 평가에 적용한 계리적가정과 동일한 가정을 사용하며, 공시이율은 회사의 과거 실제 공시이율을 적용한다.
- ⒎ 항목별 금액은 아래의 기준으로하며, 수재보험계약도 원수보험계약에 준하여 작성한다.

      |항목|금액 기준|
      |--|--|
      | **발생손해액** | 원보험계약의 보험사고로 발생한 손해액으로 <br/>지급보험금에 개별추산액과 미보고발생손해액(IBNR) 증감액을 합산한 금액 |
      | **해약환급금** | 중도해지시 보험약관에 따라 지급한 금액 |
      | **만기환급금** | 보험기간의 중도 또는 만료시 보험약관에 따라 지급한 금액 |
      | **연금지급금** | 연금지급 개시시 보험약관에 따라 지급한 금액 |
      | **계약체결비** | 신계약비 및 대리점수수료 등 계약체결로 인해 지급되는 직·간접비용을 의미하며,<br/> 회사가 합리적으로 배분하여 계리적가정 산출의 기초가 되는 금액(단, 사업비 가정 산출시 제외한 비용은 제외) |
      | **계약유지비** | 실제사업비 중 유지비 항목을 의미하며,<br/> 회사가 합리적으로 배분하여 계리적가정 산출의 기초가 되는 금액(단, 사업비 가정 산출시 제외한 비용은 제외) |
      | **손해조사비** | 장래손해조사비를 포함한 손해조사업무와 관련된 사업비를 의미하며,<br/> 회사가 합리적으로 배분하여 계리적가정 산출의 기초가 되는 금액(단, 사업비 가정 산출시 제외한 비용은 제외) |
      | **투자관리비** | 투자관련 부문에 종사하는 임직원의 급여 등 재산관리업무와 관련된 사업비를 의미하며,<br/> 회사가 합리적으로 배분하여 계리적가정 산출의 기초가 되는 금액(단, 사업비 가정 산출시 제외한 비용은 제외) |
      | **일회성비용** | 사업비 가정 산출시 제외한 비용 |
      | **출재손해액** | 출재보험금-출재보험금환입-구상이익+수입손해조사비 |

- ⒏ 위 항목 외에 `기타지급금`은 작성 대상에서 제외한다.
