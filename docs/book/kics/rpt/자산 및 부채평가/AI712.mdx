# 장기손해보험 갱신보험료 가정 (AI712)

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">
- ⒈ 지배회사 기준으로 작성하며, `실적손해율`은 연도말 기준 직전1년 연간손해율로 작성한다.
- ⒉ `실적손해율`은 위험보험료 대비 발생손해액으로 정의하며 원수기준으로 작성한다.<br/>
   위험보험료는 계약자적립액 산출 시 기초율을 적용한 위험보험료이며, 발생손해액은 지급보험금에 개별추산액과 미보고발생손해액(IBNR)의 증감액을 합산한 금액을 의미한다. 이 때 미보고발생손해액(IBNR)은 각 담보에 합리적으로 배분한 결과를 사용한다.
- ⒊ '`이익계약`' 과 '`손실계약`'은 갱신 직전 보험기간의 손해율을 기준으로 구분하며, 이익/손실에 따른 목표손해율을 구분하지 않는 경우 동일한 값을 입력한다.
- ⒋ 실손보험 중 상해입원/상해통원/질병입원/질병통원으로 분류가 불가한 보험은 `기타`로 구분한다.
- ⒌ 갱신보험료 조정 한도 산출주기가 연간인 경우 연도말 기준으로 작성할 수 있다.
- ⒍ 경험통계 `조정률 평균`은 연단위로 환산하여 기입한다.
- ⒎ 경험통계의 `통계기간`은 회사에서 정한 기준에 따라 작성한다.
- ⒏ 해당사항이 없는 부분은 공란으로 한다.
