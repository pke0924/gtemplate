# 생명보험·장기손해보험 공시이율 가정 (AI713)

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">
- ⒈ 지배회사 기준으로 작성한다.
- ⒉ 회사가 평가시점에 시장에서 적용 중인 공시이율을 산출하는 기초로 적용 공시이율(실적)을 작성하며, K-ICS (미래) 공시이율 가정도 같은 기준으로 작성한다.
- ⒊ `공시이율 카테고리`는 회사가 공시 중인 공시이율 구분단위명을 입력한다.
- ⒋ `공시이율 구분 KEY`는 평가시점에 회사가 시장에서 적용하는 공시이율 구분을 누락없이 입력 (양식에서 구분되지 않는 KEY는 `기타구분`에 입력)

    | 상품군 | 배당여부 | 이율보증기간 |
    |:---:|:---:|:---:|
    | 구분없음 | 구분없음 | 구분없음 |
    | 보장성 | 무배당 | 3개월 |
    | 저축 | 유배당 | 6개월 |
    | 연금 | 　 | 1년 |
    | 연금저축 | 　 | 3년 |
    | … | 　 | … |


- ⒌ `투자관리비용률(%)`은 자산운용이익률시나리오에서 차감되는 비용률을 뜻함
- ⒍ `(미래)조정률(%)`은 경영자행동 요소로 회사가 합리적으로 산출하되, 조정률 상/하한 한도를 적용 받는 경우에는 한도 내에서 결정한다.
- ⒎ `해약환급금`은 보험업감독규정 제7-66조 및 제7-69조에 따라 계약자적립액에서 해약공제액을 공제하여 계산한 금액이다. 단, 보험업감독규정 제7-66조제4항에 따른 보험상품의 경우에도 제1항을 준용하여 계산한다.
