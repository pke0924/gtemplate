# 생명보험·장기손해보험 보험계약대출 가정 (AI707)

<Accordion multiple activeIndex={[]}>
<AccordionTab header="[작성 요령]">
- ⒈ 지배회사 기준으로 작성한다.
- ⒉ 해당 업무보고서의 작성은 회사별 가정산출주기로 하며, 적용가정에 변경사항이 없는 경우 전기말과 동일하게 제출한다.
- ⒊ `직전3년 약관대출 이용률 최대값`: 각 보험계약대출 평가 산출단위 별로 산출한 해약환급금 대비 계약대출잔액의 직전 3년 월별 비율 중 최대값을 입력
- ⒋ `미래 약관대출 이용률 평균`: 각 보험계약대출 평가 산출단위 별로 산출한 만기별(=장래현금흐름 시점별) 미래 해약환급금 대비 미래 계약대출잔액의 평균값을 입력 (`미래 계약대출잔액 합계÷미래 해약환급금 합계`)
- ⒌ `미래 약관대출 이용률 최대값`: 각 보험계약대출 평가 산출단위 별로 산출한 만기별(=장래현금흐름 시점별) 미래 해약환급금 대비 미래 계약대출잔액의 비율 중 최대값을 입력
- ⒍ `가산이율(한도미적용)`: 상품군의 보험계약대출금액 비중으로 가중평균하여 작성
- ⒎ `가산이율(한도적용)` : `가산이율(한도미적용)`, `감독원장이 제시한 가산이율`, 그리고 `회사의 현재평가시점 가산이율` 중 작은 비율을 입력
- ⒏ `[보험계약대출잔액]`,`[해약식준비금]`에 변액 특별계정의 보험계약대출을 포함하며, 가정산출 대상상품 기준으로 집계
