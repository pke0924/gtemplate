# Ⅱ. 채권 발행시장
# 2. 구조

## 라. 투자자

채권투자자는 채권발행시장에서 모집·매출되는 채권의 청약에 응하여 발행자가 발행하는 채권을 취득하는 자로서, 자금의 공급자입니다. 채권투자자는 전문적인 지식과 대규모의 자금을 운용하고 있어 법인형태를 취하고 있는 기관투자자와 개인자격으로 자산운용을 목적으로 채권에 투자하는 개인투자자가 있는데, 우리나라에서는 기관투자자의 비중이 높습니다.
