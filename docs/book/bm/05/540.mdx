# 제4절. 채권 신규상장 신청 시 입력사항

## 가. 기본사항 

채권의 신규 상장 시 신청인은 우선 기본적인 사항을 입력함으로써 채권의 전체적인 속성을 기록합니다. 세부 사항은 신청자의 편의를 위하여 속성별로 해당사항을 클릭할 경우 나타나는 팝업창에서 입력할 수 있습니다.

<GCallout severity="success">
**상장신청 방법**

*•채권상장신청시스템：http：//bonds.krx.co.kr
*•신청기관(채권발행인) ID 및 비밀번호
    *- 신청자 ID：발행기관 고유코드 발급 후 채권상장신청시스템 신규회원가입 시 직접 설정
        *※ 발행기관 고유코드가 없는 경우 거래소 표준코드시스템(http://isin.krx.co.kr)에서 발행기관신규등록을 하여 발행기관코드 발급
    *- 비밀번호：채권상장신청시스템 신규회원가입 승인 시 임시비밀번호 발송, 로그인 후 변경
        *※ 분실 시 비밀번호재발급 신청(시스템상 신청 → Fax 발송 → 승인 → 임시비밀번호 발송) 
*•비상장채권 목록확인
    *- 표준코드가 부여된 채권은 채권상장시스템 채권목록에서 비상장채권 종목명을 선택하여 상장신청하고 코드가 부여되지 않은 경우 신규상장을 선택하여 정보입력 후 상장신청 
</GCallout>

*•발행기관 로그인 : 채권을 발행하는 기관의 아이디와 패스워드로 로그인하며, 최초 신청 시에는 발행기관 신규등록 후, 신규회원 가입신청
    *※ 발행기관 신규신청 시 사업자등록증 및 정관 제출
    *※ 신규회원 가입신청 : 상장계약서 및 회사의 개황 제출 (유동화법인일 경우 회사의 개황 면제)
    *※ 아이디는 있으나 비밀번호를 분실한 경우, 비밀번호 재신청 (재발급신청서 및 인감증명서 제출)
*•상장희망채권 : 표준코드를 미리 신청한 경우는 비상장채권 목록에서 선택하여 신청하며 표준코드가 없을 시 내용 전체 입력
*•한글(영문)종목명：한글은 40자 이내, 영문 및 숫자는 80자 이내 입력 가능 
*•상장예정일：증권신고서 제출의 경우 발행일, 일괄신고서류 제출의 경우 발행일 익일
    *※ 실제 상장일은 필요서류 제출 및 수수료 납부 등의 사유로 인해 연기될 수 있음
*•발행 및 상장금액 : 일반적으로 발행과 동시에 상장하는 경우, 발행금액과 상장금액은 동일
    *※ 상장 이후 부분 중도상환을 하는 경우 발행금액과 상장금액은 달라질 수 있음 
*•주권관련사채권：전환사채, (비)분리형 신주인수권부사채, 교환사채일 경우 선택
    *※ 분리형 신주인수권부사채일 경우 주권관련사채가 아닌 일반사채로 선택하되, 특이발행조건란에 분리형 신주인수권부사채권임을 표시 
*•이자지급방법：고정금리형/변동금리형, 이표/복리/단리/할인/복5단2/기타 중 선택
*•발행방법 : 공모 간접발행 및 공모 직접발행 중 선택(사모발행은 상장 불가)
*•표면이자율 확정여부 및 표면이자율：상장신청 시 표면이자율이 미확정일 경우, 신청일 현재이율을 입력하고 발행일 전일 (표면이자율이 최종 확정 시 재신고하고 관련 공문 또는 증권신고서 사본 제출)
*•만기상환율：일반채권은 대부분 100%이나, 전환사채 등과 같이 표면이자율과 만기보장수익률이 다를 경우 100% 이상이 될 수 있음
*•발행가액 : 일반적인 경우 100%로 입력 
*•상환방법 : 만기일시상환/분할상환(원금균등/원리금균등/불균등) 선택 
*•옵션：발행회사 또는 사채권자가 조기상환 청구할 수 있을 경우 콜/풋/콜&풋 중 선택
*•청산 시 채권 변제순위 : 선순위/중순위/후순위/후후순위 중 선택하고, 계약서에 특별한 내용이 없을 경우 선순위 선택
*•유동화：자산유동화법에 의해 설립된 유한회사가 발행한 증권만이 유동화 여부에 체크 (ASB/MBS/SLBS, 채권형/수익증권형 각각 선택)
*•신종자본증권 여부 선택 
*•조건부자본증권 유형 : 주식전환형/상각형 중 선택 
*•매출형태 : 채권의 매출일이 발행일 이전일 경우 선순위, 이후일 경우 후순위, 동일할 경우 비매출 선택(선/후매출 시 매출일 기재) 
*•최초 이자지급일：이표채인 경우에만 입력
*•영구채에 준하는 만기구조：하이브리드채권은 YES, 이외의 채권은 NO
*•특이발행조건：정형화된 항목 값 이외 채권가격에 영향을 미칠 사항을 한글 30자 이내 입력 
*•기타 참고사항 : 채권자에게 알려야 하는 특이발행조건을 100자 이내 입력

## 나. 발행조건

* **□ 이자 관련 사항**

*•이자 선/후급：이자의 선지급 또는 후지급 여부 입력 (국내발행 채권 대부분이 후급) 
*•이자 원단위 미만처리：인수기관 등과 협약에 의해 결정되며, 대부분 원단위 미만 절사
*•이자계산/지급단위월수
    *- 이표채：이자계산단위 및 이자지급단위를 동일하게 입력(3개월 이표 시 3개월 입력) 
    *- 복리채：이자계산단위는 이자 복리계산 개월 수, 이자지급단위는 만기까지 개월 수 입력
        *예) 3년만기, 6개월 복리 이자계산 시, 이자계산단위：6개월, 이자지급단위：36개월 입력
*•단리채：이자계산단위는 이자계산 개월 수, 이자지급단위는 만기까지 개월 수 입력 
    *- 할인채：이자계산단위는 공란, 이자지급단위에 만기까지 개월 수 입력 
    *- 분할상환채：이자계산단위는 이자계산 개월 수, 이자지급단위는 원금거치기간 입력
        *예) 10년만기(5년거치, 5년분할) 원금거치기간 12개월 복리채 시, 이자계산단위：12개월, 이자지급단위：60개월 입력
*•변동금리 세부사항 입력 
    *- 표면이자율결정 기준금리 : 기준금리를 입력. 주로 CD 91일물 최종호가, USD 3M Libor 등
    *- 가산금리 : +/- 가산금리를 입력 
    *- 결정시점 : 오전/오후 선택
    *- 표면이자율 결정 기준일 : 최초 표면이자율 결정 기준일 입력 
    *- 이자율결정 기준일 : 매 이자마다 변동되는 표면이자율 결정일 입력

* **□ 주권 관련 사항**

*•권리구분 : 전환권, 신주인수권(분리형/비분리형), 교환권(분리형/비분리형), 이익참가권 선택
*•대상증권명 : 대상 주권 입력 
*•행사개시일/행사종료일 
*•납입방법/청구기간 : 대용/현금납입 선택, 청구기관 등 입력 

* **□ 유동화 관련 사항**

*•유동화 관련 기관 입력 : 자산보유자, 자산보유자 유형, 자산관리자, 업무수탁자 입력
*•유동화 자산 사항 입력 : 유동화자산 유형, 유동화계획 발행총액/총평가액, 신용보강방법 입력 

* **□ 옵션 관련 사항**

*•Call/Put옵션의 차수별 행사기간, 청구 주체, 사유 입력 

* **□ 분할상환 관련 사항**

*•원금균등/원리금균등/불균등 중 선택
    *- 균등 : 균등상환액, 거치기간(개월), 거치기간 이자(단리/복리/이표 중 택1), 분할상환횟수
    *- 불균등 : 차수별 이자지급일, 상환금액, 이자율, 거치기간 이자 입력

## 다. 관련기관

*•채권 발행과 관련된 기관을 입력
    *- 등록기관 : 채권의 등록기관을 기록
    *- 원리금지급대행기관 : 원리금 지급대행 계약을 맺은 은행
    *- 대표주관회사
    *- 수탁기관/사채관리회사 
    *- 보증기관
    *- 신용평가기관 및 신용등급 : NICE신용평가, 한국신용평가, 한국기업평가, 서울신용정보 등
    *- 인수기관 및 인수금액 (4사까지 입력 가능)

## 라. 채권 CFI (Classification of Financial Instruments)

국제표준화기구(ISO)에서 금융상품 속성별 분류를 위해 공통된 용어와 표준화된 분류기준을 정하여 국제적으로 동일하게 적용하기 위해 사용하는 코드로서 채권의 대략적인 속성을 나타냅니다.

## 마. 신청인 사항

발행기관 및 대표주관회사의 담당자를 각각 입력하고 세부적으로는 신청인구분(발행기관/대표주관회사), 성명, 전화, 이메일 등을 기재(전화번호는 직장번호 기록)합니다.

## 바. 기타 유의사항

위와 같이 신청시스템에서 입력한 후 신청자는 이를 증빙할 수 있는 서류를 제출해야 합니다. 이중 증권신고서는 신청결과조회화면의 파일업로드에서 .dsd 파일을 선택하고 파일설명을 한글 25자 이내로 입력 후 전송합니다. 이외 기타 제출 서류는 공문의 형태로 팩스 송부(051-630-2751) 또는 파일업로드의 방법으로 제출합니다.

또한, 채권상장시스템은 신규상장 이외에 채권의 발행 후 발생할 수 있는 채권 관련 모든 사항을 신고할 수 있으며, 발행회사의 채권 담당자는 신고 의무가 있습니다(상장계약서에 포함).
