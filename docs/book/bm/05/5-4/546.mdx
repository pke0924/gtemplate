# 4. 채권 신규상장 신청 시 입력사항

## 바. 기타 유의사항

위와 같이 신청시스템에서 입력한 후 신청자는 이를 증빙할 수 있는 서류를 제출해야 합니다. 이중 증권신고서는 신청결과조회화면의 파일업로드에서 .dsd 파일을 선택하고 파일설명을 한글 25자 이내로 입력 후 전송합니다. 이외 기타 제출 서류는 공문의 형태로 팩스 송부(051-630-2751) 또는 파일업로드의 방법으로 제출합니다. 

|국내채권<br/>(상장규정 시행세칙 제15조)|외국채권<br/>(상장규정 시행세칙 제17조)|
|:------------------------------------|:------------------------------------|
|1. 견양<br/>2. 신탁증서, 인수 또는 매출계약서, 모집위탁계약서(간접발행)<br/>3. 사업설명서<br/>4. 최근 3사업년도 재무제표 및 당해 사업년도 반기재무제표<br/>5. 최근 사업년도 감사보고서<br/>6. 납입증명서<br/>7. 기타 필요서류 (최초 상장 시 상장계약서, 회사의 개황 등)|1. 정관 또는 이에 상당하는 서류<br/>2. 신탁증서, 인수 또는 매출계약서, 모집위탁계약서(간접발행)<br/>3. 상장대리인선임 계약서 사본<br/>4. 최근 3사업년도 감사보고서<br/>5. 신용평가서 사본<br/>6. 기타 필요서류 (최초 상장 시 상장계약서, 회사의 개황 등)|
|※ 효력이 발생한 증권신고서 사본 제출 시 : 2\~5 생략<br/>※ 국채, 지방채, 통안채 : 2\~7 생략<br/>※ 모집발행지방채, 특수채(신고서 면제) : 3\~7 생략<br/>※ 보증사채 : 4\~5 생략|※ 효력이 발생한 증권신고서사본 제출 시 1\~6 생략|

또한, 채권상장시스템은 신규상장 이외에 채권의 발행 후 발생할 수 있는 채권 관련 모든 사항을 신고할 수 있으며, 발행회사의 채권 담당자는 신고 의무가 있습니다(상장계약서에 포함).

|변경 사항|신고할 내용|
|:------:|:----------|
|발행회사 관련 사항|1. 상호변경<br/>2. 합병/분할|
|채권금액 관련 사항|1. 추가상장<br/>2. 중도상환(부분/전액)<br/>3. 권리행사(주권관련/일반채권)<br/>4. 발행가액 변경|
|금리 관련 사항|1. 표면 이자율 변경<br/>2. 변동금리 변경|
|채권자 권리 관련 사항|1. 권리행사가액 변경<br/>2. 상환일 변경|
|기타 사항|위에서 정의되지 않은 사항|
