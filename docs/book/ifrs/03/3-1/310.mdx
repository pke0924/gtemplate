# 3.1. 최초인식

보험계약집합은  보장 기간이 시작될 때, 보험계약자가 첫 번째 보험료를 지급해야 할 때, 집합이 손실부담계약집합이 되는 때 중 가장 이른 시점에 최초 인식해야 함. 다만, 기준서에서는 보험계약집합이 속해있는 보험계약 포트폴리오에 직접 귀속되는 보험계약집합의 판매, 인수심사 및 개시 관련 원가에서 생기는 현금흐름인 보험취득 현금흐름을 보험계약집합을 인식하기 전에 지급한 경우에는 별도의 인식기준을 규정하고 있음.
